Ventoy Install – Clean Start

Working commands for Fedora / Linux
(USB device: /dev/sda)

⚠ Important: These commands will erase everything on the USB drive (/dev/sda). Make sure you have selected the correct device.
1
Go to home folder and clean up old files
cd ~
rm -rf ventoy*
2
Download the latest Ventoy (Linux version)
wget https://sourceforge.net/projects/ventoy/files/v1.1.16/ventoy-1.1.16-linux.tar.gz
3
Extract the archive
tar -xzf ventoy-1.1.16-linux.tar.gz
4
Enter the Ventoy folder
cd ventoy-1.1.16
5
Confirm the script exists
ls

You should see Ventoy2Disk.sh in the list.

6
Double-check the USB device
lsblk

Confirm that your USB drive is still showing as sda (the 58.6G one).

7
Install Ventoy
sudo sh Ventoy2Disk.sh -i -g -s /dev/sda

-i = install  |  -g = GPT partition style  |  -s = Secure Boot support

Type y and press Enter when it asks for confirmation.

✓ Done!
After the install finishes, the USB is ready.
Just copy your Windows ISO (and any other ISOs) onto the large partition.

After Installation

You can now copy ISO files like this:

cp /path/to/Windows.iso /run/media/$USER/Ventoy/

Or simply open the USB in the file manager and drag & drop the ISO files.

Useful Commands Later

Update Ventoy (keeps your ISOs):

sudo sh Ventoy2Disk.sh -u /dev/sda

Force reinstall:

sudo sh Ventoy2Disk.sh -I -g -s /dev/sda
Commands tested on Fedora · Official site: ventoy.net
