Ventoy Guide

Create a multi-boot USB drive from Linux
(Perfect for Windows + Linux ISOs)

What is Ventoy?

Ventoy is a free open-source tool that turns a USB drive into a multi-boot device. You install it once, then simply copy ISO files onto the drive. When you boot from the USB, Ventoy shows a menu and lets you choose which ISO to boot.

No need to reformat the drive every time you want a different operating system. It works great with Windows, Linux, WinPE, and many other images.

0 Prerequisites

  • A Linux computer
  • A USB drive (8 GB or larger recommended)
  • A Windows ISO file (or any other ISO you want)
  • Root / sudo access
⚠ Warning: Installing Ventoy will erase everything currently on the USB drive. Back up any important files first.

1 Download Ventoy

Go to the official website and download the latest Linux package:

https://www.ventoy.net/en/download.html

Look for the file named something like:
ventoy-1.1.16-linux.tar.gz

You can also download it directly with a terminal command (example for version 1.1.16):

wget https://sourceforge.net/projects/ventoy/files/v1.1.16/ventoy-1.1.16-linux.tar.gz

2 Extract the archive

tar -xzf ventoy-*-linux.tar.gz
cd ventoy-*-linux

3 Identify your USB drive

Plug in the USB drive, then run:

lsblk

Look for your USB device. It is usually something like /dev/sdb or /dev/sdc (never use /dev/sda — that is almost always your main hard drive).

Double-check the device name! Choosing the wrong disk will erase the wrong drive.

4 Install Ventoy on the USB

Run the following command (replace /dev/sdX with your actual USB device):

sudo ./Ventoy2Disk.sh -i /dev/sdX

Useful options

  • -i → Install Ventoy (fails if already installed)
  • -I → Force reinstall
  • -u → Update existing Ventoy installation
  • -g → Use GPT partition style (recommended for modern UEFI systems)
  • -s → Enable Secure Boot support

Example with GPT + Secure Boot:

sudo ./Ventoy2Disk.sh -i -g -s /dev/sdX
After installation finishes, the USB will have two partitions. The larger one is where you will copy your ISO files.

5 Copy your ISO files

Mount the large partition of the USB (it is usually mounted automatically) and simply copy your ISO files onto it: 

cp /path/to/Windows.iso /media/youruser/Ventoy/
cp /path/to/ubuntu.iso  /media/youruser/Ventoy/

You can put the files in any folder — Ventoy will find them. You can also keep regular files on the same drive; they will not interfere.

6 Boot from the USB

  1. Restart your computer
  2. Enter the boot menu (usually F12, F10, Esc, or Del — depends on the manufacturer)
  3. Select the USB drive
  4. Ventoy’s menu will appear listing all your ISO files
  5. Choose the Windows (or Linux) ISO and press Enter

That’s it!

The selected operating system will boot directly from the ISO. You can now install Windows, try a live Linux distro, or run rescue tools — all from the same USB.

Extra Tips

  • You can add or remove ISO files at any time — just copy or delete them.
  • No need to reinstall Ventoy when you want a new ISO.
  • Ventoy supports Windows 7 → 11, Windows Server, WinPE, and almost all major Linux distributions.
  • To update Ventoy later, use: sudo ./Ventoy2Disk.sh -u /dev/sdX
  • Official documentation: ventoy.net/en/doc_start.html
